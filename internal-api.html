Internal API

Macros

ttt_calloc(T, n)

Macro for allocating memory. This macro calls C standard library function calloc() and casts the returned pointer as a pointer to type T.

Parameters:

  • T – Type of the memory to be allocated.

  • n – Number of elements of type T to be allocated.

Returns:

Pointer to the allocated memory.

ttt_free(p)

Macro for freeing memory. This macro calls ttt_free_() function.

Parameters:

  • p – Pointer to the memory to be freed.

ttt_assert(COND, MSG)

Macro for asserting a condition. This macro calls ttt_assert_() function.

Parameters:

  • COND – Condition to be asserted.

  • MSG – Message to be printed if the condition is not met.

Functions

void ttt_free_(void **ptr)

Function for freeing memory. ttt_free_() calls C standard library function free() and sets the pointer to NULL.

Parameters:

ptr – Pointer to the memory address (pointer to a pointer) to be freed.

void ttt_log(const ttt_log_type_t verbose, const char *fmt, ...)

Print a debug message if the verbosity level is greater than 0.

Parameters:

  • verbose – Verbosity level.

  • fmt – Format string.

  • ... – Format string arguments.

void ttt_assert_(int cond, const char *msg, const char *file, const unsigned line)

Assert a condition to be true and print a message if the assertion fails. Use the macro ttt_assert() instead of using this function directly.

Parameters:

  • cond – Condition to be asserted.

  • msg – Message to be printed if the assertion fails.

  • file – Name of the file where the assertion is made.

  • line – Line number where the assertion is made.
